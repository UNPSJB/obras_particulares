Baja de tramites¶

Esta opcion permite visualizar en forma de lista los tramites a dar de baja dentro del sistema. Las opciones de visualizacion son las siguientes:

  • nro: Indica el numero de tramite.
  • medidas: Indica las medidas del tramite.
  • tipo: Indica el tipo de tramite.
  • estado: Indica el estado actual del tramite.
  • profesional: Indica el profesional del tramite.
  • propietario: Indica el propietario del tramite.
  • dar de baja: Puede eliminar un tramite haciendo click en el boton
