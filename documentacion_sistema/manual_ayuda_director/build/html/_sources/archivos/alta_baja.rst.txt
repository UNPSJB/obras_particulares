Alta Baja de usuarios
=====================

Esta opcion permite dar alta o baja a usuarios dentro del sistema. Las opciones a visualizar son las siguientes:

- **ID**: Indica el id del usuario.
- **Email**: Indica el email de un usuario.
- **Staff**: Indica si el usuario es staff del sistema.
- **Estado**: Indica el estado del usuario. Si el usuario se encuentra activo o inactivo dentro del sistema.
- **Nombre de Usuario**: Indica el nombre de usuario.

Puede confirmar el alta o baja de un usuario haciendo click en el boton alta o baja respectivamente.
