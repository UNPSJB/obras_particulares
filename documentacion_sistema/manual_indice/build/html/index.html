Manual de Inicio¶

Bienvenido al manual de ayuda al ingreso al sistema. En esta pantalla encontrara dos funcionaliades principales:

Ingreso al sistema¶

Para ingresar al sistema debera proporcionar los siguientes datos en los campos:

  • : El nombre de usuario que se le haya sido otorgado.

  • : La contraseña que haya seleccionado para su usuario actual.

Una vez ingresados los datos debera hacer click en la opcion

Creacion de un nuevo profesional¶

Para solicitar el alta de un nuevo profesional en el registro de profesionales del sistema debera completar el formulario de alta de profesionales indicando los siguientes campos:

  • : Es el documento del profesional.

  • : Es el nombre del profesional.

  • : Es el apellido del profesional.

  • : Es el telefono del profesional.

  • : Es el domicilio del profesional.

  • : Es el cuil del profesional. Tenga consideracion del formato especificado XX-XXXXXXXX/X

  • : Es el mail del profesional. Tenga consideracion del formato especificado XXXXXX@XXX.com

  • : Es el numero de matricula habilitante del profesional.

  • : Indica la profesion actual del profesional ingresado.

  • : Indica la categoria que posee el profesional siendo posibles las siguientes opciones:
  • : Campo disponible para la carga de una copia del certificado de profesional habilitado.

Una vez completado el formulario en su totalidad podra solicitar el alta de un nuevo profesional ingresando la opcion , en caso de querer reestablecer los datos del formulario puede ingresar la opcion
